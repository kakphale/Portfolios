        HTML: HTML stands for Hyper Text Markup Language. 
        It is used to design the front end portion of web pages using markup language.
        HTML is the combination of Hypertext and Markup language. Hypertext defines the link between the web pages.
        The markup language is used to define the text documentation within tag which defines the structure of web pages.
        CSS: Cascading Style Sheets, fondly referred to as CSS,
        is a simply designed language intended to simplify the process of making web pages presentable.
        CSS allows you to apply styles to web pages. 
        More importantly, CSS enables you to do this independent of the HTML that makes up each web page.
